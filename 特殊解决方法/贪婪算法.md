## 例题

1221. 分割平衡字符串

```
class Solution:
    def balancedStringSplit(self, s: str) -> int:
        res = 0
        d = 0
        for ch in s:
            if ch == "L":
                d += 1
            if ch == "R":
                d -= 1
            if d == 0:
                res += 1
        return res

```
